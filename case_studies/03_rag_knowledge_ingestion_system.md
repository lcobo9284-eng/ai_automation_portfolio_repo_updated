# RAG Knowledge Ingestion System

## Project summary

Designed and supported a knowledge ingestion system that added documents, audio, and video content into a vector database to power retrieval augmented generation workflows.

## Business challenge

The team needed a scalable way to transform files from cloud storage into searchable knowledge for AI assistants. Content included documents and media files that required extraction, transcription, chunking, embedding, and storage.

## Solution approach

The system retrieved files from Dropbox, processed documents and media, generated embeddings, stored content in Pinecone, and tracked processed files in operational databases or spreadsheets.

## Core capabilities

• Dropbox file retrieval  
• Document processing  
• Audio and video transcription support  
• Text chunking  
• OpenAI embeddings  
• Pinecone vector storage  
• Airtable and Google Sheets tracking  
• Scheduled or agent triggered ingestion  

## Tools used
