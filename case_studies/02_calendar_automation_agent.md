# Calendar Automation Agent

## Project summary

Built and maintained a calendar automation system capable of interpreting scheduling requests, managing events, creating meetings with attendees, and coordinating calendar actions through AI assisted workflows.

## Business challenge

Calendar operations required manual handling of meeting creation, attendee coordination, event updates, meeting details, and event deletion. The goal was to create a more efficient assistant that could understand user requests and execute calendar related tasks through connected tools.

## Solution approach

The workflow used an AI agent to interpret user intent and delegate actions to specialized sub workflows. These actions included retrieving access tokens, creating events, updating meetings, deleting events, and coordinating meeting information.

## Core capabilities

• Natural language calendar task interpretation  
• Event creation with attendees  
• Calendar event deletion  
• Meeting detail extraction  
• Access token retrieval  
• Integration with Microsoft Outlook and Zoom related tools  
• Structured output parsing for scheduling data  

## Tools used

• n8n  
• OpenAI models  
• Microsoft Outlook integrations  
• Zoom related tools  
• HTTP requests  
• Structured output parsing  
• JSON
