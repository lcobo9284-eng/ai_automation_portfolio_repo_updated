# Murphy AI Operations Assistant

## Project summary

Supported a multi tool AI assistant ecosystem that connected messaging channels, AI models, vector databases, workflow tools, and communication platforms to automate operational support tasks.

## Business challenge

Operational teams needed a conversational assistant that could receive user inputs, interpret requests, call specialized tools, retrieve knowledge, and return useful responses across connected communication channels.

## Solution approach

The system used AI models, memory, vector retrieval, workflow tools, and messaging integrations to process requests and delegate tasks to specialized automations.

## Core capabilities

• Conversational AI assistant logic  
• WhatsApp and Telegram related interactions  
• AI model orchestration  
• Supabase or vector based knowledge retrieval  
• Tool based workflow execution  
• Message formatting and response delivery  
• Multi workflow architecture  

## Tools used

• n8n  
• OpenAI models  
• Supabase  
• WhatsApp  
• Telegram  
• HTTP requests  
• Vector stores  
• AI agent tools  
• JSON  

## My contribution
