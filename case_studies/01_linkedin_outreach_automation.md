# LinkedIn Outreach Automation System

## Project summary

Designed and maintained an AI powered LinkedIn outreach automation system to support lead generation, prospect research, personalized messaging, and campaign follow up. The system connected prospecting tools, AI agents, Google Sheets, and external automation services to reduce manual outreach work and improve consistency across campaign execution.

## Business challenge

Manual prospecting and outreach required repetitive work across multiple platforms. The process involved defining target audiences, extracting leads, researching companies, writing personalized messages, creating follow ups, and tracking campaign outputs.

## Solution approach

The automation transformed a natural language audience description into a structured prospecting workflow. It generated search logic, extracted lead information, created company analysis, produced personalized outreach messages, and organized follow up sequences.

## Core capabilities

• Audience criteria interpretation  
• Apollo search URL generation  
• Lead extraction using Apify and PhantomBuster related workflows  
• AI generated company analysis  
• Personalized LinkedIn connection messages  
• Follow up message generation  
• Google Sheets tracking  
• Multi step orchestration through n8n AI agents  

## Tools used

• n8n  
• OpenAI models  
• Apify  
• PhantomBuster  
• Apollo  
• Google Sheets  
• HTTP requests  
• JSON  
• AI agents
