# Confidentiality Notice

This repository was created as a professional portfolio and does not contain proprietary code, private workflows, client data, credentials, API keys, tokens, webhook URLs, database identifiers, internal prompts, or confidential business information.

## What is included

• Anonymized project descriptions  
• High level architecture summaries  
• Safe documentation of tools and workflow logic  
• Recreated explanations based on general automation patterns  
• Portfolio templates for future documentation  

## What is intentionally excluded

• Original n8n workflow JSON files  
• Credentials and connected account data  
• API keys, access tokens, webhook IDs, and secrets  
• Client names connected to specific confidential workflows unless already approved for public use  
• Internal prompts and proprietary business logic  
• Raw datasets, lead lists, email addresses, phone numbers, and private documents  

## Why this matters

Automation workflows often contain sensitive information inside nodes, credentials, webhook URLs, headers, test data, pinned data, prompts, and API requests. Publishing raw workflow exports can expose private systems and create legal, security, and confidentiality risks.

## Publication standard

Before publishing any project publicly, each item should be reviewed to confirm that it is either fully anonymized, recreated with fictitious data, or independently developed for demonstration purposes.
